FAQs

How can I link CSS and JS file in my code?

This is a very simple task, from the installation section Just copy the CDN link of CSS and paste it inside head tag and copy the CDN link of JS and paste it before the closing </body> in your .html file.

Can I made changes in the components by writing CSS code?

Yes! You can make changes to it but if you overwrite things then you have to write !important keyword in your css code.

Can I add more CSS property?

Yes, you can add more custom CSS properties by targeting the classname or ID of that component.
